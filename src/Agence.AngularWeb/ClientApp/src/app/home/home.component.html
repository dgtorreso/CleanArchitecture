Hola, Mundo!

Bienvenido a este ejercicio de consultas. Esta aplicación está desarrollada con:

ASP.NET Core y C# para el lado del servidor, usando la arquitectura de CleanArchitecture con los siguientes componentes:

  • Core: contiene la lógica del servicio y la definición de la base de datos.
  • Infrastructure: Contiene la configuración necesaria para el acceso a datos, incluyendo las referencias para que la aplicación funcione con MSSqlServer y MySql con Entity Framework.
  • AngularWeb: Contiene la lógica de la presentación de la aplicación, utiliza WebApi para prestar servicios al cliente. 

Angular y TypeScript en el lado de cliente.

Materialize CSS y el tema Free Material Design Admin Template

 

Para usar el sistema elige la opción Comercial Performance y juega con los filtros para obtener la data.
